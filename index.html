API Overview

The Indonesian Maps API provides comprehensive access to geographical and administrative data for Indonesia, including provinces, cities, districts, villages, and islands.

Complete Geographic Data

Access detailed information about all administrative levels in Indonesia, from provinces down to villages.

Demographic Information

Get population data, area measurements, and other demographic statistics for each region.

Search & Filter

Advanced search functionality with pagination and hierarchical filtering capabilities.

Base URL
https://api.sukmaaji.my.id/indonesian-maps
Response Format
JSON UTF-8
Authentication
None Required
Rate Limiting
No Limits

API Documentation

Complete reference for all available endpoints

Provinces (Provinsi)

Available: Geographic coordinates and boundary path data are available for province endpoints.
GET /indonesian-maps/provinsi Get all provinces
Query Parameters
Parameter Type Description
page integer Page number (default: 1)
limit integer Items per page (default: 50, max: 1000)
search string Search by province name
Example Response
{
  "success": true,
  "data": [
    {
      "kode_provinsi": 32,
      "nama_provinsi": "Jawa Barat"
    }
  ],
  "message": "Successfully retrieved 38 provinces",
  "total": 38,
  "page": 1,
  "limit": 50,
  "totalPages": 1
}
GET /indonesian-maps/provinsi/{code} Get province detail
Path Parameters
Parameter Type Description
code integer Province code (e.g., 32 for West Java)
Example Response
{
  "success": true,
  "data": {
    "kode_provinsi": 32,
    "nama_provinsi": "Jawa Barat"
  },
  "message": "Successfully retrieved province Jawa Barat"
}
GET /indonesian-maps/provinsi/{code}/geo Get province geographical data
Example Response
{
  "success": true,
  "data": {
    "kode_provinsi": "32",
    "nama_provinsi": "Jawa Barat",
    "ibukota": "Bandung",
    "lat": "-6.902224715926122",
    "lng": "107.6187597542088",
    "elv": "739.0",
    "tz": "7",
    "luas": "37053.331",
    "penduduk": "51316378.0",
    "status": "1"
  },
  "message": "Successfully retrieved geo data for province Jawa Barat"
}

Regencies/Cities (Kabupaten/Kota)

Available: Geographic coordinates and boundary path data are available for city/regency endpoints.
GET /indonesian-maps/kabupaten-kota Get all regencies/cities
Query Parameters
Parameter Type Description
kode_provinsi integer Filter by province code
search string Search by name
page integer Page number
limit integer Items per page
Example Response
{
  "success": true,
  "data": [
    {
      "kode_kabupaten_kota": "32.04",
      "nama_kabupaten_kota": "Kabupaten Bandung"
    }
  ],
  "message": "Successfully retrieved regencies/cities",
  "total": 27,
  "page": 1,
  "limit": 10,
  "totalPages": 3
}

Districts (Kecamatan)

Note: Geographic coordinates and boundary data are not yet available for district endpoints.
GET /indonesian-maps/kecamatan Get all districts
Query Parameters
Parameter Type Description
kode_kabupaten_kota string Filter by regency/city code
search string Search by district name
Example Usage
GET /indonesian-maps/kecamatan?kode_kabupaten_kota=32.04

Villages (Desa/Kelurahan)

Note: Geographic coordinates and boundary data are not yet available for village endpoints.
GET /indonesian-maps/desa-kelurahan Get all villages
Query Parameters
Parameter Type Description
kode_kecamatan string Filter by district code
search string Search by village name
Example Usage
GET /indonesian-maps/desa-kelurahan?kode_kecamatan=32.04.01

Islands (Pulau)

GET /indonesian-maps/pulau Get all islands
